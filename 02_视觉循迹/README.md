# 02 视觉循迹

基于图像处理的循迹方案，通过摄像头识别轨道线完成路径规划与控制。

---

## 目录结构

```
02_视觉循迹/
├── 01_图像采集与预处理/   # 摄像头初始化、二值化、降噪
├── 02_线段检测/           # 边缘提取、质心/拟合算法
├── 03_控制算法/           # 图像偏差 → PID → 电机输出
├── 04_调试工具/           # 上位机图像显示、串口调试
└── README.md
```

---

## 方案概述

| 模块 | 技术选型 | 说明 |
|------|----------|------|
| 摄像头 | OV7725 / OpenMV / 总钻风 | 视野广，可预判弯道 |
| 图像处理 | 大津法二值化 + 质心检测 | 低算力需求，实时性好 |
| 控制 | 位置式 PID | 图像偏差量作为反馈 |

---

## 核心流程

```
摄像头采集 → 灰度化 → 二值化 → ROI 裁剪
    → 扫线/质心计算 → 偏差提取 → PID → 差速控制
```

---

## 开发环境

- 摄像头：OV7725（MT9V032 / OpenMV 可选）
- 主控：TMS320F28xxx / STM32
- IDE：CCS / Keil
