# 图像采集与预处理

## 摄像头初始化

- 分辨率建议：`188×120` 或 `320×240`（视帧率需求调整）
- 接口：DVP 并口 / SPI（视主控平台）
- 曝光与增益：固定曝光优先，避免运动模糊

## 预处理流程

```
原始图像 → 灰度化 → 高斯/均值滤波 → 大津法/固定阈值二值化 → ROI 截取
```

## 关键参数

| 参数 | 建议值 | 说明 |
|------|--------|------|
| ROI 起始行 | 帧高 40%~60% | 只处理前方感兴趣区域 |
| 二值化阈值 | 动态（大津法）| 适应光线变化 |
| 滤波核大小 | 3×3 | 去除椒盐噪声 |

## 文件说明

- `src/camera_init.c` — 摄像头初始化代码
- `src/image_proc.c` — 图像预处理函数
