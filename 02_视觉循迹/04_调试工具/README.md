# 调试工具

辅助调试的上位机工具与串口协议说明。

## 串口调试协议

| 字段 | 字节数 | 说明 |
|------|--------|------|
| 帧头 | 2B | `0xAA 0xBB` |
| 偏差值 | 2B | int16，像素偏差 |
| 左速 | 2B | 左电机 PWM |
| 右速 | 2B | 右电机 PWM |
| 帧尾 | 1B | `0xCC` |

## 上位机推荐

- **VOFA+**：波形显示 PID 效果
- **匿名上位机**：姿态与数据可视化
- **串口助手**：原始数据查看

## 图像回传（可选）

- 通过 USB 或 WIFI 将处理后的二值图像传回上位机
- 推荐使用 OpenMV IDE 内置图像传输功能

## 文件说明

- `tools/` — 上位机脚本（Python / Qt）
