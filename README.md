## 仓库结构

```
├── 01_灰度循迹/          # 灰度传感器循迹及多路升级
├── 02_视觉循迹/          # 基于摄像头的图像处理循迹
├── 03_视觉跟踪云台/       # 双轴云台视觉目标跟踪
└── 04_资料收集/          # 开源案例、教程、CCS配置、往年赛题
```

## 模块简介

| 模块 | 方案 | 状态 |
|------|------|------|
| **灰度循迹** | 多路灰度阵列 + PID 差速控制，支持直道加速与弯道减速 | 🔧 进行中 |
| **视觉循迹** | OV7725/OpenMV + 大津法二值化 + 质心扫线 + PID | 🔧 进行中 |
| **视觉跟踪云台** | OpenMV 目标检测 + 串口通信 + 双轴舵机 PID 闭环 | 🔧 进行中 |

## 开发环境

- **主控**：TMS320F28379D / STM32（视题目）
- **IDE**：Code Composer Studio（CCS）12.x / Keil MDK
- **视觉**：OpenMV IDE / K210 MaixPy
- **版本管理**：Git + GitHub

1. 克隆仓库：`git clone <repo-url>`
2. 根据选用主控，进入对应模块目录查阅 README
3. 开发环境配置参考 [04_资料收集/03_CCS配置教程](./04_资料收集/03_CCS配置教程/README.md)
