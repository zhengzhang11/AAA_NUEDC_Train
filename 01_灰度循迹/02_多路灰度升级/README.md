# 多路灰度升级

采用多路灰度阵列（如 8~16 路）结合增量式 PID，提升弯道处理能力与抗干扰性。

## 升级要点

1. **更宽视野**：增加传感器路数，覆盖更宽轨道范围
2. **增量式 PID**：减少积分饱和，提升动态响应
3. **交叉点处理**：识别十字路口并决策
4. **速度自适应**：直道加速、弯道减速

## 算法优化方向

- 加权质心算法替代简单阈值判断
- 引入微分项改善超调
- 黑白场景自适应阈值
