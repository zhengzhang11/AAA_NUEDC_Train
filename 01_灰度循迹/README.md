# 01 灰度循迹

基于灰度传感器的循迹方案，从基础实现到多传感器融合升级。

---

## 目录结构

```
01_灰度循迹/
├── 01_基础循迹/          # 单排灰度传感器 + PID 控制
├── 02_多路灰度升级/       # 多路灰度阵列，提升抗干扰与弯道性能
├── 03_PID调参记录/        # 各阶段 PID 参数记录与调试日志
└── README.md
```

---

## 方案概述

| 阶段 | 方案 | 说明 |
|------|------|------|
| 基础版 | 单排灰度 + 位置式 PID | 直道稳定，弯道适中 |
| 升级版 | 多路灰度阵列 + 增量式 PID | 更宽视野，弯道响应更快 |

---

## 开发环境

- MCU：TMS320F28xxx / STM32（待定）
- IDE：CCS / Keil
- 传感器：灰度传感器模块（模拟/数字输出）
