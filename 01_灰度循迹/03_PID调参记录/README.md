## 调参经验

- **Kp 过大**：震荡，蛇形走线
- **Kd 过大**：高频抖动，对噪声敏感
- **Ki 过大**：积分饱和，响应迟滞
- 建议先整定 Kp，再加 Kd，最后微调 Ki
