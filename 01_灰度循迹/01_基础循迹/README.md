# 基础循迹

单排灰度传感器 + 位置式 PID 控制实现基础循迹。

## 硬件连接

| 传感器引脚 | MCU引脚 | 说明 |
|-----------|---------|------|
| OUT1~OUTn | ADC/GPIO | 灰度值采集 |
| VCC       | 3.3V/5V | 供电 |
| GND       | GND     | 共地 |

## 算法流程

```
采集灰度值 → 计算加权偏差 → PID 运算 → 差速输出 → 电机驱动
```

## 文件说明

- `src/` — 源码目录
- `docs/` — 接线图、原理图

## 备注

- 传感器数量建议 5~8 路
- 安装高度参考：距地面 10~15 mm
