# 电机

常见适用于窄底盘小车的电机：

| 产品           | MG513XL    | MG513X     | MG310电机  |
| -------------- | ---------- | ---------- | ---------- |
| 减速比         | 1:28       | 1:28       | 1:20       |
| 额定电压       | 12V        | 12V        | 7.4V       |
| 额定电流       | 1A         | 0.36A      | ≤200mA    |
| 堵转电流       | 6A         | 3.2A       | ≤2.0A     |
| 减速后空载转速 | 370±12rpm | 370±12rpm | 500±13rpm |
| 减速后额定转速 | 370±12rpm | 370±12rpm | 500±13rpm |
| 额定扭矩       | 3kgf.cm    | 1kgf.cm    | 0.4kgf.cm  |
| 堵转扭矩       | 3kgf.cm    | 3kgf.cm    | 3kgf.cm    |
| 功率           | 10W        | 4W         | 1.85W      |

**建议采用 MG513X 配合 60mm 左右的硅胶轮（轮子直径由6mm径轴决定，可购买的最小直径为60mm）** 。这套动力总成的机械上限非常高，能完美承载后续编写的复杂多级 PID 或串级控制算法。不升级XL的原因为：在绝大部分纯平面的循迹、迷宫或避障任务中，标准版 MG513X 的动力已经处于**严重过剩**的状态。盲目追求XL更大的扭矩不仅会突破轮胎的物理抓地力极限导致打滑、影响PID闭环控制系统，还会因为电机更长、更重而增加底盘的转动惯量，让小车在连续过弯时变得笨重迟钝。

MG513X对比JGB37-520：这两款电机都属于 **同一个重量级（37mm外径减速箱，500系电机）** 。它们都能提供远超普通小车所需的充沛扭矩，但在**机械精度**上存在决定性的差异。**JGB37-520** ：采用标准的大批量偏心正齿轮减速箱，它的齿轮啮合间隙较大；**MG513X**通常针对智能车竞赛做过齿轮组优化（部分版本使用精度更高的斜齿或专门调配的齿隙），机械虚位明显小得多，控制响应更加线性，闭环系统的超调量更容易被压制。
