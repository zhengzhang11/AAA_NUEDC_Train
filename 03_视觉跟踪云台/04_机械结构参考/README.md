# 机械结构参考

云台安装、选型与调试注意事项a。

## 云台选型

| 类型 | 说明 |
|------|------|
| 2-DOF 舵机云台 | 结构简单，适合轻量摄像头（OpenMV、小型USB摄像头） |
| 无刷云台（BGC） | 精度高、稳定性强，适合赛场高精度跟踪需求 |

## 安装注意

1. **重心平衡**：安装摄像头后调整配重，减轻舵机负载
2. **零位标定**：上电后先让云台回到机械中位
3. **线缆管理**：舵机线留足余量，避免转动受限
4. **安装高度**：云台距地高度影响视野，根据目标大小调整

## 结构示意

```
        摄像头
          |
     [Pitch 舵机]  ← 俯仰轴
          |
     [Yaw 舵机]    ← 水平旋转轴
          |
       底座固定
```

## 推荐器材

- 舵机：MG996R（大扭矩）、SG90（轻量场景）
- 云台支架：铝合金 2-DOF 云台套件
- 摄像头：OpenMV H7 Plus / K210 摄像头模块
