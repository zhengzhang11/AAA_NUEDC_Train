# 03 视觉跟踪云台

基于图像处理的目标跟踪云台，通过识别目标的位置偏差驱动舵机或电机完成双轴（Pitch / Yaw）方向对准。

---

## 目录结构

```
03_视觉跟踪云台/
├── 01_目标识别/           # 颜色/形状检测，输出目标坐标
├── 02_云台控制/           # 舵机驱动 + 双轴 PID 控制
├── 03_通信接口/           # 上位机与下位机数据交换
├── 04_机械结构参考/       # 云台安装示意图与注意事项
└── README.md
```

---

## 方案概述

| 模块 | 技术选型 | 说明 |
|------|----------|------|
| 视觉 | OpenMV / 树莓派 + OpenCV | 识别目标，输出 (cx, cy) |
| 云台驱动 | 舵机（MG996R 等） | 双轴 2-DOF 云台 |
| 控制 | 双轴独立 PID | Yaw/Pitch 分别闭环 |

---

## 核心流程

```
摄像头采集 → 目标检测 → 获取 (cx, cy)
    → 计算偏差 (ex, ey) → 双轴 PID → 舵机 PWM 输出
```

---

## 开发环境

- 视觉模块：OpenMV H7 / K210 / 树莓派
- 主控：STM32 / TMS320F28xxx
- 舵机：标准 PWM 舵机（50Hz，500~2500μs）
