# 直流弱信号前置放大与采集电路设计 (基于 LM358P & STM32F407)

本设计旨在采集 0~100 mV 的直流微弱信号，通过 LM358P 进行前置缓冲、滤波与放大，最终送入 STM32F407 的 ADC 进行高频 (10 kHz) 采样。

## 一、硬件电路设计 (LM358P 前置放大)

### 1. 电路原理图 (ASCII Art)

```text
输入端 (0~100 mV DC)
      │
      ├── R_prot (100 Ω) ──┐   保护电阻
      │                    │
      │          R_f (10 kΩ)── 低通滤波
      │                    │
      │                   (B)──C1 (1 μF)──GND   fc=15.9 Hz, 50Hz衰减~68%
      │                    │
      │              U1A(+) [LM358 Pin 3]
      │             U1A: 电压跟随器（缓冲）
      │              U1A(-) [Pin 2] ◄─── U1A out [Pin 1]
      │
      └── (GND reference)

U1A out [Pin 1] ──────────────── U1B(+) [Pin 5]

                         U1B: 同相放大器，增益 = 21
                         U1B(-) [Pin 6] ──┬── Rg (1 kΩ) ── GND
                                          └── Rf (20 kΩ) ──┤
                                U1B out [Pin 7] ──────────── 此点

U1B out ── R_out (100 Ω) ──────── PA5 (ADC1_IN5)
                              │
                           C2 (100 pF) ── GND  (高频抗混叠)

电源:  VCC [LM358 Pin 8] = 3.3V
       GND [LM358 Pin 4] = GND
       C_bypass (0.1 μF) between Pin8 and GND (紧靠IC)
```


### 3. 元件清单 (BOM)

| **元件标号** | **值** | **作用**                                       |
| ------------------ | ------------ | ---------------------------------------------------- |
| U1                 | LM358P       | 双路运算放大器                                       |
| R1 (R_prot)        | 100 Ω       | 输入保护电阻                                         |
| R2 (R_f)           | 10 kΩ       | 一级低通滤波电阻                                     |
| C1 (C_filter)      | 1 μF        | 一级低通滤波电容 (**$f_c=15.9 \text{ Hz}$**) |
| R3 (Rg)            | 1 kΩ        | 放大器增益下臂电阻                                   |
| R4 (Rf)            | 20 kΩ       | 放大器增益上臂电阻                                   |
| R5 (R_out)         | 100 Ω       | ADC 保护与容性负载隔离电阻                           |
| C2 (C_hf)          | 100 pF       | 高频抗混叠电容                                       |
| C3 (C_bypass)      | 0.1 μF      | VCC 去耦电容 (靠近 Pin 8)                            |

## 二、STM32F407 连接定义

该工程的 ADC 采样方案是  **DMA + 普通单通道** ， **没有用硬件过采样** ，具体配置如下：

| 项目     | 配置                                                                                  |
| -------- | ------------------------------------------------------------------------------------- |
| 采样方式 | **DMA 循环模式** （Circular），非过采样                                         |
| ADC 通道 | 单通道：ADC1 CH5（PA5）                                                               |
| 分辨率   | **12 位** （`ADC_RESOLUTION_12B`），原始值范围 0~4095                         |
| 触发源   | TIM3 TRGO（Update 事件），10 kHz 定时触发，非连续/软件触发                            |
| 采样时间 | `ADC_SAMPLETIME_480CYCLES`（最大值，降低噪声）                                      |
| DMA      | DMA2 Stream4 CH0，Halfword，Circular，TC 中断                                         |
| 缓冲区   | `g_adc_dc_buf[1000]`，对应 100 ms / 5 个 50 Hz 周期                                 |
| "过采样" | **软件均值** （`adc_dc_mean_raw()` 对 1000 个样本求平均），非硬件 16 位过采样 |

**结论：** 硬件层面是标准 12 位单通道 ADC + DMA，通过对 1000 个样本软件求均值来提升有效分辨率，并非 STM32 硬件过采样寄存器（STM32F407 本身也不支持硬件过采样，该功能从 STM32G0/L5 系列起才有）。

| **信号** | **引脚** | **功能**                                   |
| -------------- | -------------- | ------------------------------------------------ |
| 模拟输入       | PA5            | ADC1_IN5 (接收放大后的 0~2.1V 信号)              |
| 内部触发       | TIM3_TRGO      | 配置为 10 kHz 触发 ADC 采样 (无外部引脚物理连接) |
